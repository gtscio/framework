# GTSC Framework Core

This package contains helper methods and classes which implement some of the fundamental features required for a framework.

- Guards
- Type Checking
- Error Handling
- Validation
- Encoding
- Conversion
- Localisation

## Installation

```shell
npm install @gtsc/core
```
